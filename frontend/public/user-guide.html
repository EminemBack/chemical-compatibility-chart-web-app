Kinross Chemical Compatibility — End‑User Guide

Create a Compatibility Request

Follow these steps to submit a request with optional attachments, contact info, and GPS location.

  1. Open the App
  2. Fill all the required fields
  3. Attach Evidence
  4. Review your product in Safety Data Sheet
  5. Provide Context
  6. Check the Compatibility Matrix
  7. Check the box & Submit the Request for Review

1) Open the App

Open the application in your browser. Use the URL provided by your team or IT.

Use a modern browser (Chrome, Edge, or Firefox) for the best experience.

2) Fill all the required fields

Provide necessary informations about the request.

3) Attach Evidence

Upload supporting files/pictures that help validate your request (e.g., photos).

  • Use the “Attach file(s)” or “Upload” button where available.
  • Accepted file types and size limits may apply. If an upload fails, try a smaller file or PDF.
  • Click on ℹ️ to have an idea of the picture you should upload.
Do not upload sensitive or confidential documents unless policy explicitly permits.

4) Review your product in Safety Data Sheet

  • Search by name, identifier, or category.
  • Add multiple items if the evaluation involves more than one chemical/material.

5) Provide Context

Select the different Hazard existing in your container:

  • Select one or more hazards.
  • Go down and provide the separation distance.

6) Check the Compatibility Matrix

To know about the combination and the separation distance of your existing Hazard

If permission is denied or unavailable, enter coordinates manually.

7) Check the box & Submit the Request for Review

Review your entries, then click “Submit” to send your request for review.

  • Requests typically enter “Pending Review” until an admin approves or requests rework.
  • If rework is requested, you will see feedback and can edit and resubmit.
You can return later to check status and reviewer notes.

For help or questions, contact your site administrator.

Brand note: Header bar and accents use Kinross‑inspired gold and charcoal. Replace CSS variables with official Kinross hex values if needed.
